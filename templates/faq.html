Needlee V1

How do I activate Needlee V1 drone?

Connect your mobile device (connected to the internet) with the Needlee Console, login using credentials your recieved with an email after the order. Then power Needlee V1 to connect via Needlee Console. Follow the instructions in the Console to activate Needlee V1.

Before using Needlee V1, what app do I need to install?

Needlee V1 can only be used with the Needlee Console. You can download the latest PWA application on your device or use the Needlee Console web version. Needlee Console requires stable internet connection.

What is the wind speed resistance of Needlee V1?

The max wind speed resistance of Needlee V1 is 8 m/s.

Is Needlee V1 waterproof?

No. Avoid contact with water during use as it may lead to damage of the device. If it rains during flight, please return and land as soon as possible, and wait until the aircraft is fully dry inside before using it again.

How do I update Needlee V1 drone software?

Each update is available in your Needlee Console.

Needlee Console

How do I get a Needlee Console account?

Needlee Console account is automatically created for each Needlee V1 drone. You recieve temporary login and password via an email with full setup instructions after your Needlee V1 order is created.

What do I need to use Needlee Console?

The easiest way is to use the web version of Needlee Console. You can additionlly install the PWA Needlee Console application to increase performance. You'll need stable internet connection to use Needlee Console.

Where do I get Needlee Console tutorials?

You'll recieve a complite user guide via an email with full setup instructions after your Needlee V1 order is created.
